# Dice Master (GM and Player tool)

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/license/mit/)

This is a D&D session management tool for both game masters and players.
It allows you to roll dice, manage characters, and keep track of game sessions.

## Features

- **Dice Rolling**: Roll various types of dice with customizable options.
- **Character Management**: Create and manage characters with detailed stats.
- **Session Tracking**: Keep track of game sessions, including notes and events.
- **User-Friendly Interface**: Designed for ease of use during gameplay.
- **Cross-Platform**: Built with Flutter, works on both iOS and Android.
- **Open Source**: Free to use and modify under the MIT License.
- **Community Driven**: Contributions are welcome! Check out
  the [Contributing Guide](CONTRIBUTING.md).
- **Documentation**: Comprehensive documentation available in the [docs](docs/.) directory.
- **Testing**: Includes unit tests and integration tests to ensure reliability.
- **CI/CD**: Continuous integration and deployment setup for automated testing and builds.
- **Localization**: Supports multiple languages for a wider audience.
- **Accessibility**: Designed with accessibility in mind to support all players.
- **Dark Mode**: Supports dark mode for comfortable viewing in low light conditions.
- **Customizable Themes**: Change the look and feel of the app with themes.
- **Notifications**: Get reminders for game sessions and events.
- **Backup and Restore**: Easily backup your game data and restore it when needed.
- **Community Support**: Join our community for help and discussions.
- **Regular Updates**: Actively maintained with regular updates and new features.
- **Analytics**: Track usage and performance with built-in analytics.

## Installation

To install and run the Dice Master app, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/maxstain/dice_master.git
    ```
2. Navigate to the project directory:
   ```bash
   cd dice_master
   ``` 
3. Install dependencies:
   ```bash
    flutter pub get
    ```
4. Run the app:
    ```bash
        flutter run
    ```
5. For web support, ensure you have the Flutter web environment set up and run:
    ```bash
    flutter run -d chrome
    ```
6. For desktop support, ensure you have the Flutter desktop environment set up and run:
    ```bash
    flutter run -d windows
    ```
7. For mobile support, ensure you have the Flutter mobile environment set up and run:
    ```bash
    flutter run -d android
    ```
